# العمارة التقنية

## المكونات الرئيسية

```
┌─────────────────────────────────────────────────────────────┐
│                        الزبون                               │
│              يتصل على رقم Twilio                             │
└──────────────────────┬──────────────────────────────────────┘
                       │
                       ▼
┌─────────────────────────────────────────────────────────────┐
│                      Twilio                                 │
│            استقبال وتوجيه المكالمات                         │
└──────────────────────┬──────────────────────────────────────┘
                       │ Webhook: /call /respond
                       ▼
┌─────────────────────────────────────────────────────────────┐
│                    Flask Backend                            │
│         • معالجة الصوت والمحادثات                           │
│         • التكامل مع OpenAI/Whisper                        │
│         • إدارة الطلبات                                    │
└──────────────┬──────────────────────┬──────────────────────┘
               │                      │
               ▼                      ▼
    ┌──────────────────┐    ┌──────────────────┐
    │   PostgreSQL     │    │   OpenAI/Whisper │
    │   • Calls        │    │   • Speech to Text│
    │   • Messages     │    │   • Text to Speech│
    │   • Orders       │    │   • GPT-4        │
    └──────────────────┘    └──────────────────┘
```

## تدفق المكالمة

```
1. الزبون يتصل على رقم Twilio
   ↓
2. Twilio يرسل POST /call
   ↓
3. نحن ننشئ سجل مكالمة في قاعدة البيانات
   ↓
4. نحول النص الترحيب لصوت (TTS)
   ↓
5. نشغّل الصوت والاستماع للرد (Gather)
   ↓
6. الزبون يتكلم → Twilio يصرف POST /respond
   ↓
7. نحول الصوت لنص (Whisper)
   ↓
8. نرسل للـ GPT-4 مع السياق
   ↓
9. GPT يرد بالنص
   ↓
10. نحول الرد لصوت
   ↓
11. نشغّل الصوت والاستماع مرة أخرى
   ↓
12. تكرار 6-11 حتى انتهاء المحادثة
```

## تكامل الخدمات

### Twilio
- استقبال مكالمات واردة
- تشغيل ملفات صوتية
- تسجيل الحوار
- Webhooks للـ Backend

### OpenAI
- **Whisper**: تحويل الصوت لنص (Speech-to-Text)
- **GPT-4o**: معالجة اللغة الطبيعية (NLP)
- **TTS**: تحويل النص لصوت (Text-to-Speech)

### ElevenLabs (اختياري)
- جودة صوت أعلى من OpenAI TTS
- أصوات عربية متعددة

### PostgreSQL
- تخزين دائم للمكالمات والرسائل
- تتبع الطلبات والإيرادات
- تقارير وإحصائيات

## الأمان

- **Token Bearer**: حماية API
- **CORS**: السماح بطلبات من النطاقات المصرحة فقط
- **Logging**: تسجيل جميع الأحداث
- **Environment Variables**: حفظ بيانات حساسة

## التوسع

### المستقبل
- قاعدة بيانات إضافية (Redis للـ Cache)
- WebSocket للتحديثات الحية
- Admin Panel متقدم
- تحليلات متقدمة (Analytics)
- دعم عملات مختلفة
